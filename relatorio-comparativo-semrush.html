Metodologia

1) Coleta das top 30 keywords organicas de matbrindes.com.br no banco BR da Semrush.
2) Cruzamento das mesmas keywords com os arquivos lacos-keywords-posicoes.xlsx (Lacos) e somarcas-keywords-posicoes.xlsx (So Marcas).
3) No cruzamento entre dominios, as keywords foram normalizadas (minusculas + remocao de acentos) para evitar erro de correspondencia textual.
4) Quando a keyword nao aparece para o dominio no recorte consultado, a posicao foi marcada como N/R.
5) Para trafego, foi usado Organic Traffic do endpoint de overview no banco BR.
